# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates and maintenance.

| Version                 | Supported |
| ----------------------- | --------- |
| Latest Release          | ✅         |
| Previous Stable Release | ✅         |
| Older Versions          | ❌         |

Only the latest maintained versions receive security updates. Users are strongly encouraged to keep their installations up to date.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly and privately.

### How to Report

Send a detailed report containing:

* Description of the vulnerability
* Steps to reproduce
* Potential impact
* Screenshots, logs, or proof-of-concept (if applicable)

Reports can be submitted through:

* GitHub Security Advisories (preferred)
* Private communication with the project owner

### Response Timeline

After receiving a valid report:

* Initial acknowledgment: within 7 days
* Investigation and assessment: within 30 days
* Security patch release: depending on severity and complexity

### Responsible Disclosure

Please do not publicly disclose vulnerabilities until a fix has been released or permission has been granted by the project owner.
